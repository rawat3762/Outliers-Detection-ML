# 📊 Outliers-Detection-ML

## 📌 Description

A simple machine learning project demonstrating how to detect outliers in data using the IQR (Interquartile Range) method.

---

## 📌 What are Outliers?

Outliers are data points that are very different from the rest of the data.

---

## ⚙️ Workflow

* Import required libraries (pandas, matplotlib, seaborn)  
* Load dataset  
* Apply IQR method to detect outliers  
* Visualize data using boxplot & displot 

---

## 🚀 Features

* Detects outliers using IQR method  
* Uses boxplot & displot for visualization  
* Beginner-friendly implementation  
* Clean and simple workflow  

---

## 🛠️ Technologies Used

* Python  
* Pandas    
* Matplotlib  
* Seaborn  
* Jupyter Notebook
